mode

Description:

public EditMode mode { set; get; }

Specifies mode which widget is handling. Note that this can be changed at any time and widget will be swapped with new one that handles that mode. At this point there are two options, if widget was bound then all is good, if value was handled directly assigning correct value is solely responsability of application.
