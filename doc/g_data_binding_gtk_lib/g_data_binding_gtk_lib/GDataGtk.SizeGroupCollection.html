SizeGroupCollection

Object Hierarchy:

Description:

public class SizeGroupCollection : Object

Provides collection of named sizegroups. This collection self manages its lifetime based on lifetime of ref_owner specified on creation. Since it has its own memory management only weak references should be ever held

Since:

0.1
