AutoValueModeWidget.with_type

Description:

public AutoValueModeWidget.with_type (Type value_type, EditMode mode = VIEW, CreationEditMode creation_mode = FULL, DefaultWidgets? default_widgets = null, SizeGroupCollection? size_control = null)

Creates new AutoValueModeWidget for specified value type

Parameters:

value_type

Value type

mode

Starting mode (VIEW/EDIT)

creation_mode

Creation mode description which can be VIEW, EDIT or both.

default_widgets

Specifies widget creation mechanism, this can be reset at anytime by setting default_widgets property. If specified value is null, then its value will be resolved trough DefaultWidgets.get_default()
