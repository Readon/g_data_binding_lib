GtkBuildableMapper

Object Hierarchy:

Description:

public class GtkBuildableMapper : Object, BinderMapper

Provides easy mapping with composite widgets or layouts loaded from Glade In case of this mapper target should be parent container or composite widget which contains widgets specifying correct Gtk.Buildable names (ID in Glade)

Since:

0.1
