hold

Description:

public BindingPointer hold (BindingPointer pointer)

Helper method to hold pointer reference without the need to either store it in storage or be assigned as local variable

This is useful for middle links of the chain as when holder gets unreferenced, all pointers being held by it get same treatment. If there is a need to undo it, method release() should be used

Parameters:

pointer

Pointer being held

Returns:

Held pointer reference for convenience of possibility to chain API in objective languages
