value_objects_changed

Description:

public signal void value_objects_changed (CustomPropertyNotificationBindingSource object, ContractChangeType change_type)

Signal is sent whenever value object is added or removed. Value change is never signaled here. For state change it is better to rely on "state" property value and binding to the object

Parameters:

object

Object being added or removed

change_type

Type of change
