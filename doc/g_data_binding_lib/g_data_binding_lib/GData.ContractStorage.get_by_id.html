get_by_id

Description:

public static BindingContract? get_by_id (int id)

Searches for contract by specified unique id which is autoassigned on pointer creation (accessible trough its id property)

This search is not limited to pointers stored in storage. Any pointer can be accessed

Parameters:

id

Id of requested pointer
