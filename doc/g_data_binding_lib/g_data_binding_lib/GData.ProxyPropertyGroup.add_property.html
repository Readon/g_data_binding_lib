add_property

Description:

public ProxyProperty? add_property (string name, Object obj, string property_name, Value invalid_value, bool bidirectional = true, Type restrict_to = INVALID)

Creates proxy property to tracking one single property and allowing that tracking to respect pointer/contract source changes

Parameters:

obj

Object or pointer/contract that owns property

property_name

Property name

invalid_value

Specifies value which should be returned on momentary invalid state machine (aka. source being pointer/contract but pointing to null)

bidirectional

Specifies if state machines are bidirectional or not

restrict_to

Type safety. If property type is not transformable to that type it will act like connection never happened

property

Name under which property is stored, duplicates are not supported
