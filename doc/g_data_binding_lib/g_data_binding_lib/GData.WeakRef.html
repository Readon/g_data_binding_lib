WeakRef

Object Hierarchy:

Description:

public class WeakRef

.Net C# Non-generic WeakRef like implementation

Main purpose is always having safe minimal object that can be used in hash tables or arrays and that object always being valid no matter its contents

Difference from .Net implementation is that this one supports setting new target in order to minimize object creation needs.

Note that this class is not derived from GLib.Object and as such it is not possible to connect to property notification or track validity of pointed object. For that purpose StrictWeakReference should be used
