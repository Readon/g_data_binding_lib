BindingReferenceType

Description:

public enum BindingReferenceType

Specifies how BindingPointer should handle reference on data that it points to

Since:

0.1


Namespace: GData
Package: g_data_binding_lib

Content:

Enum values:

  • DEFAULT - In normal cases this means WEAK but certain parts need different operation of that and handle it differently (binding pointer can resolve as contract reference type in order to have uniform handling)
  • WEAK - Default way of handling references and also preferred. Only use strong when there is no other way
  • STRONG - Binding adds strong reference on data objects for the duration of activity. This requires binding to be either suspended or disposed in order to release the reference it holds over source and target object
