GObjectArray

Object Hierarchy:

Description:

public class GObjectArray : Object, ListModel

GObjectArray is holding list of non generic objects and emits signals when objects are added or removed.

Second trade it provides is also guarantee of unique objects in list if specified on creation.

Due to the fact that it implements GLib.ListModel it is automatically usable as model for Gtk.ListBox or Gtk.FlowBox

Since:

0.1
