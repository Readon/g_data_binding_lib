EnumStateGroup

Description:

public EnumStateGroup (Object obj, string property_name, bool bidirectional = true, int invalid_value = 0)

Creates new EnumStateGroup with no state machines. To add state machine call get_state() which either resolves already existing state machine or creates new one

Parameters:

obj

Object or pointer/contract that owns property

property_name

Property name

bidirectional

Specifies if state machines are bidirectional or not

invalid_value

Specifies value which should be returned on momentary invalid state machine (aka. source being pointer/contract but pointing to null)
