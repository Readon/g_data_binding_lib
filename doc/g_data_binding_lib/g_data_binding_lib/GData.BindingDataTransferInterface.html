Content:

Properties:

  • public abstract bool is_valid { get; }
    Specifies if transfer data is valid or not. It can be invalid for multiple reasons. - It was only invoked for introspection - Data is not correct
  • public abstract bool is_introspectable { get; }
    Specifies if data transfer is at least introspectable if not valid
  • public string property_name { owned get; }
    Provides name as bindable point when redirection is done to this object it self
  • public string property_nick { owned get; }
    Provides nick as bindable point when redirection is done to this object it self
  • public string property_blurb { owned get; }
    Provides blurb as bindable point when redirection is done to this object it self

Methods:

  • public abstract weak Object? get_object ()
    Object being handled

    Specifies type of handled object. Note that this is available even when real reference is presented and all resolving in STATIC types should be handled trough this and not object reference
  • public abstract ParamFlags get_property_flags ()
    Returns parameter flags for property
  • public abstract string get_object_type_name ()
    Returns string representation of handled object type which is useful to distinct when non GObject type is wrapped inside to get real type name

    Returns property name
  • public abstract string get_nick ()

    Resolves data value in property of object
  • public abstract void set_value (Value val)
    Sets data value in property of object
  • public abstract void connect_signal ()

  • public abstract void disconnect_signal ()

    Returns true if reference to object is valid, false if not
