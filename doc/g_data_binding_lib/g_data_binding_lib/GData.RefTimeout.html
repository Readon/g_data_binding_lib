RefTimeout

Object Hierarchy:

Description:

public class RefTimeout : Object

RefTimeout is almost the same as GLib.Timeout with one exception of adding safety layer to its lifetime

This makes it guarateed to stop if specified reference is not alive anymore

Since:

0.1
