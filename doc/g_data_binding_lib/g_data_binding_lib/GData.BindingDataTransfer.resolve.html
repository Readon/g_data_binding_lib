resolve

Description:

protected virtual void resolve (string property_name)

Most important method in derived classes. This is invoked as part of construction and at this point object should get to know what it needs to handle data and signaling

Object reference is already accessible trough get_object() at this point if this is not introspection only object. In case of STATIC get_introspection_type() should be used instead

Signal should not be connected at this point as binding invokes connect_signal() and disconnect_signal() when needed

Parameters:

property_name

Property name
