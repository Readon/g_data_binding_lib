set_default

Description:

public static void set_default (Binder? binder)

Sets specified Binder as default Binder. This also changes it for all contracts that don't specify their own as their default behaviour is using Binder resolved by get_default() and that value is not cached on contract

Use this to either employ debugging per need or when custom Binder is really needed

Parameters:

binder

Binder which is set default. If specified value is null then new default binder is created as most basic Binder possible
