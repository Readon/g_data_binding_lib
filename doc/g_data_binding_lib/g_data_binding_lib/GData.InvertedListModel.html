InvertedListModel

Object Hierarchy:

Description:

public class InvertedListModel : Object, ListModel

List model that inverts order of displayed items whose lifetime depends solely on lifetime of handled model and as such it shouldn't ever be referenced manually.

Note that reference holding of actual model must be maintained by application

Example: my_listbox.bind_model (new InvertedListModel(my_model), ...)
