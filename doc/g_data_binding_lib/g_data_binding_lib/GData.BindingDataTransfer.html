BindingDataTransfer

Object Hierarchy:

Description:

public class BindingDataTransfer : Object, BindingDataTransferInterface

Specifies get/set override for property value in binding. Do not store any hard references as it get passed in request and

If object being handled is not derived from GObject then some sort of wrapper should be provided

Since:

0.1

All known sub-classes:
  • GObjectBindingDataTransfer
  • GSettingsBindingDataTransfer

Namespace: GData
Package: g_data_binding_lib

Properties:

  • public virtual bool is_valid { get; }

  • public virtual bool is_introspectable { get; }
    Specifies if data transfer is at least introspectable if not valid
