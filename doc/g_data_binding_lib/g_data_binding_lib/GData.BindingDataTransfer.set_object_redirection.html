set_object_redirection

Description:

public void set_object_redirection (Object obj)

This should only be called during call to resolve and not later. What it does is specify another target of binding which is useful when handling things like redirected properties

Example redirection is to BindingDataTransfer object it self and binding to nick or blurb properties

Parameters:

obj

Object which data handling is redirected to
