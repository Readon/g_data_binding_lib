FlagState

Description:

public FlagState (Object obj, string property_name, uint flag, bool bidirectional = true, uint invalid_value = 0)

Creates boolean flag connection between object and state property based on specific flag

If specified object is binding pointer then this connection is actively reestablished whenever source changes

Parameters:

obj

Object or pointer that contains flags

property_name

Name of property containing flags

bidirectional

Specifies if assignment goes both ways or not
