get_signal_list_from_ref

Description:

public uint[] get_signal_list_from_ref (Object? obj)

Returns signals for specified objects type.

Note that this only returns its own signals, not signals from parent classes or interfaces. If that is needed then get_full_signal_list_from_ref() should be used
