is_valid

Description:

public abstract bool is_valid { get; }

Returns if source data is valid or not.

Validity is checked first against valid source and then updated on each property value transfer where bind() specified specific way to check if data is valid or not. This way source validity always checks for correct conditions even when property bindings are added or removed. When property binding is added, so is its condition and same for removal where property binding condition is removed as well

Note that contract value of "is-valid" will be cumulative of all conditions on all properties
