What is AutoContainerValues and what is its purpose

Back to main page

AutoContainerValues is one of g_data_binding_gtk_lib auto layouting options whose main objective is to be able to create lightweight widgets that are oriented either horizontally or vertically. Most notable usage is probably auto creation and filling of ListBox models which when combined with the fact that it supports work trough SizeGroupCollection makes it very versatile widget.

It should be noted that optimization for cached rebuilds of same container is on TODO list
