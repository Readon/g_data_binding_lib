  • MasterSlaveArray - Type simplification for KeyValueArray of KeyValueArrays. Generic structure is just too complex (ObjectArray of KeyValuePair for key ObjectArray of KeyValuePairs
  • ObjectArray - ObjectArray is holding list of objects and emits signals when objects are added or removed.
  • StrictWeakReference - StrictWeakReference is different from WeakReference in two things.
  • WeakReference - .Net C# WeakReference like implementation
  • ContractArray - ContractArray is merely type definition to simplify code where used
  • CustomBindingSourceData - CustomBindingSourceData<T> is one of value objects implementations. It serves as well as example on how to create custom value objects as having usable functionality.
